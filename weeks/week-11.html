Velocity Conversions

  • 1 m/s = 2.237 mph
  • 90 mph = 40.2 m/s (typical fastball)

Angular Velocity

  • 1 RPM = 0.105 rad/s
  • 2400 RPM = 251 rad/s (typical spin)
  • ω = 2πf (frequency to angular velocity)

Day 2: Accuracy, Precision & Error

50 minutes Analytical

🎯 Key Topics

🎯

Accuracy Definition

Closeness to the true or accepted value in measurements.

📍

Precision Definition

Reproducibility and consistency in repeated measurements.

⚠️

Measurement Error

Random errors affect precision; systematic errors affect accuracy.

📊

Error Analysis

Identifying sources and minimizing impact on data quality.

🔑 Accuracy vs Precision

High Accuracy, High Precision

Ideal scenario

Measurements consistently close to true value

High Accuracy, Low Precision

Random error present

Average is correct, but scattered measurements

Low Accuracy, High Precision

Systematic error present

Consistent measurements, but offset from true value

📝 Lab Procedure: Advanced Technology Data Collection

1

Rapsodo 3.0 Setup (15 min)

Position Rapsodo 3.0 device, configure app settings, calibrate distance measurements, establish proper alignment with mound and plate.

  • Place unit 6-8 feet behind home plate
  • Align with pitcher's mound using app guidance
  • Verify environmental conditions (lighting, wind)
  • Test connectivity and data recording
2

Multi-Device Data Collection (25 min)

Simultaneously collect data using Rapsodo 3.0 and Pocket Radar for comparison analysis, documenting all environmental conditions.

  • Record 30+ pitches per pitcher
  • Document velocity, spin rate, release metrics
  • Cross-reference radar gun readings
  • Note weather conditions and ball type
3

Data Validation & Export (10 min)

Export Rapsodo data, organize measurements, perform initial quality checks, and prepare datasets for statistical analysis.

  • Export CSV data from Rapsodo app
  • Organize radar gun measurements
  • Check for measurement outliers
  • Prepare data for statistical software
